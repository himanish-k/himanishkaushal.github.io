Adaptable Game Board

A software development group project that implemented object oriented design and the model-view-controller model to create a general framework for board games that worked with a pluggable artificially intelligent component to play against. Other board games such as Othello, 'Hare and Hounds' and 'Tic Tac Toe' were built on top of this with a GUI design. All the games were tested with various AI techniques to play against including Minimax. Consistent use of the Github version control system was made. All classes were tested using JUnit 4 to ensure that the software adhered to basic standards including correct response to all types of input and performing functions in a timely manner.

    Features:
  • Java
  • Eclipse IDE
  • JUnit 4
  • Github
  • GUI
  • Model-View-Controller Pattern
  • Artificial Intelligence
  • Minimax

TFTP Transfer

A software development group project for a course that dealt with real-time concurrent systems (SYSC 3303) that made use of multi-threaded software components and transferred files from one computer to another connected to the same local network using Trivial File Transfer Protocol. The project made extensive use of Java and implemented an object oriented model to separate entities such as the client and the server. I lead the team with efficient management of time and human resources, and delivered a professionally competent product that earned the team an impressive mark on the project.

    Features:
  • Java
  • Eclipse IDE
  • Trivial File Transfer Protocol
  • Github
  • Java Socket library
  • Concurrent Processes
